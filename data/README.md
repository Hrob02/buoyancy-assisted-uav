# Shared Data

This directory holds shared datasets used across MATLAB and Python components.

## What is tracked vs ignored

| Directory | Tracked? | Description |
|---|---|---|
| `raw/` | ✅ Yes | Raw input datasets (small files only, < 500 KB) |
| `processed/` | ✅ Yes | Pre-processed reference data |
| `generated/` | ❌ No (git-ignored) | Outputs generated by scripts |

## Naming Convention

`<source>_<descriptor>_<YYYY-MM-DD>.<ext>`

Example: `lidar_canopy_scan_2024-11-15.csv`

## Large Files

For large datasets (> 500 KB), use Git LFS or store externally and document the source here.
