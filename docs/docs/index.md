# Buoyancy-Assisted UAV Documentation

Welcome to the documentation for the Buoyancy-Assisted UAV project.

## Overview

This project provides:

- **Aerodynamic & energy modelling** in MATLAB under `matlab/`.
- **ROS 2 simulation** (Python/rclpy) under `ros_ws/`.
- **Optional desktop UI** under `ui/`.
